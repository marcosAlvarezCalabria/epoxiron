- Tables: Card layout or horizontal scroll
- Forms: Full-width, stacked buttons
- Modal: Full-screen

**Tablet (640px - 1024px):** â­ PRIMARY TARGET
- Navbar: Horizontal dropdowns

- Modal: 60-70% width

- Navbar: Full horizontal
- Tables: Full width, all columns
- Forms: Max-width 800px

- Spacing: Min 8px between

- Small: 4px (inputs)

**Animations:**
- Duration: 200-300ms
